Pico Bluetooth Device Scanner - Setup and Usage 

=======Prerequisites: =========
Python 3.x installed on your system.
The following Python libraries:
- serial
- tkinter
- threading
- time

To install missing libraries, run:
pip install pyserial

======= Setting Up Your Raspberry Pi Pico W: =========
Make sure pico connected
Ensure main.py is set to run automatically on startup.
Disconnect and reconnect the Pico after uploading to ensure the code is running.

===== Running the Desktop Application:===============
- Open the terminal and navigate to the directory containing bluetoothApp.py.
- Run the application using:
	- python bluetoothApp.py
